# Exercices Git

## Comment utiliser ?

1. Assurez-vous d'avoir Python installé (vérifiez avec `python --version`)
2. Ouvrez un terminal (PowerShell, Bash, CMD...) puis naviguez jusqu'au répertoire où se trouvent les exercices (`cd chemin/vers/répertoire`)
3. Executez les scripts pour générer les répertoires de travail (`python ex01_init_setup.py` par exemple)
